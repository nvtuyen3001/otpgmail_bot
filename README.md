# NeyuT OTP Service

A secure OTP service for phone number verification.

## Features

- Phone number management
- OTP retrieval
- Admin panel
- Secure authentication

## Deployment

### Frontend (Netlify)
1. Connect GitHub repository
2. Deploy from `main` branch
3. Publish directory: `public`

### Backend (Render)
1. Connect GitHub repository
2. Deploy as Web Service
3. Set environment variables

## Usage

1. Access admin panel to manage phone numbers
2. Enter phone numbers to get OTP codes
3. Copy OTP for verification

## Security

- All sensitive data is protected
- Admin authentication required
- Secure API endpoints
